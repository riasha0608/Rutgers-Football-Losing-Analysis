# Rutgers-Football-Losing-Analysis
Our Project aims to analyze why the Rutgers Scarlet Knights Football team finished the 2025 season with a losing 5-7 record. Using game statistics, player data, and schedule performance data sets, we will explore what factors are contributing most to Rutgers’ losses during the 2025 season.

Each of the files are labeled with the step of the data science pipeline that they are a part of. Running each of the files in a systematic manner (ingestion, cleaning, feature_engineering, queryanalysis, eda, visualizations, machinelearningmodels) will result in similar results as we have discussed in our final report for the project.
